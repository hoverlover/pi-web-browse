# Web Research

I'll help you research this topic by searching the web and reading relevant sources.

## Research Process

1. **Search** - I'll use `web_search` to find relevant sources
2. **Evaluate** - I'll review the search results for quality and relevance  
3. **Fetch** - I'll use `web_fetch` to read the most promising sources
4. **Synthesize** - I'll compile the findings into a comprehensive response

## Guidelines for Web Use

- Search first, then fetch - don't guess URLs
- Fetch 2-3 sources minimum for balanced perspective
- Prioritize official documentation, reputable blogs, and recent content
- Use CSS selectors when you need specific sections (e.g., `article`, `.content`)
- Note when information might be outdated (check publication dates)
